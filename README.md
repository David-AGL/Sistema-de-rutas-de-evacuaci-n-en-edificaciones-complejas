# Sistema-de-rutas-de-evacuaci-n-en-edificaciones-complejas
Aplicación que modela edificaciones complejas como grafos para generar múltiples rutas de evacuación desde la ubicación del usuario hasta salidas seguras. Permite visualizar alternativas, marcar bloqueos y recalcular caminos en tiempo real, facilitando decisiones rápidas y reduciendo riesgos en emergencias.
